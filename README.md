
# Services Inspector - Operations Management System 2026

نظام إدارة عمليات التفتيش والخدمات البيئية لعام 2026. مصمم لإدارة القوى العاملة وجدولة الزيارات الميدانية بدقة عالية.

## المميزات
- جدولة ذكية للمفتشين.
- تقارير PDF سنوية وشهرية احترافية.
- إدارة كاملة للمواقع والقوى العاملة.
- دعم كامل للغتين العربية والإنجليزية.

## التشغيل المحلي
1. قم بتثبيت التبعات:
   ```bash
   npm install
   ```
2. ابدأ وضع التطوير:
   ```bash
   npm run dev
   ```

## النشر على Vercel
1. ارفع المشروع على GitHub.
2. اربط المستودع بـ Vercel.
3. سيتم الكشف عن إعدادات Vite تلقائياً وبدء النشر.
